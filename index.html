PokeAPI Explorer – Descubre y Explora Pokémon en Tiempo Real

Aplicación web desarrollada en Angular que permite explorar Pokémon de forma rápida, intuitiva e interactiva.

Incluye un listado completo con cientos de Pokémon, un sistema de búsqueda por nombre, paginación optimizada, imágenes oficiales y una vista detallada con estadísticas, tipos, habilidades y datos relevantes de cada criatura.

La aplicación consume la API pública de PokeAPI y está diseñada con un enfoque en rendimiento, modularidad y experiencia de usuario. Cuenta con una interfaz moderna, fluida y totalmente responsive, aprovechando componentes reutilizables, servicios centralizados y buenas prácticas de Angular para garantizar una navegación ágil y una carga eficiente de datos.

La aplicación está en un servidor gratuito, por lo que puede tardar entre 30 segundos y un 1 minuto en responder.

Aplicación de Buscador de Películas

Aplicación web desarrollada para buscar, filtrar y descubrir películas en tiempo real utilizando una API externa. Ofrece un buscador dinámico que muestra resultados instantáneos, fichas detalladas de cada película.

Incluye un sistema de navegación fluida entre resultados y una interfaz optimizada para ofrecer una experiencia rápida, moderna y responsiva. El proyecto está construido con React, Vite, JavaScript y un backend ligero en Node.js, priorizando rendimiento, modularidad, buenas prácticas y una arquitectura escalable que facilita la incorporación de nuevas funcionalidades.

Puede que a la hora de buscar una película, el sistema de búsqueda no sea tan rápido como se espera, esto se debe a que la API externa a la que se conecta puede ir con retarlo o incluso no mostrar resultados.
